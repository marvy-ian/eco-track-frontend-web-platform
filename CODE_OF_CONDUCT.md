EcoTrack aims to provide a welcoming, respectful, and collaborative environment for everyone who contributes to the project.

We value contributions from people with different backgrounds, experiences, and skill levels.

## Expected Behavior

Contributors should:

* Communicate respectfully.
* Provide constructive feedback.
* Accept different opinions and approaches.
* Focus on improving the project.
* Respect other contributors and users.
* Follow project guidelines.

## Unacceptable Behavior

The following actions are not allowed:

* Harassment or discrimination.
* Personal attacks.
* Offensive comments.
* Publishing private information.
* Deliberately disrupting discussions or contributions.
* Submitting harmful or malicious code.

## Contribution Guidelines

When contributing:

* Keep discussions professional.
* Explain decisions clearly.
* Respect code reviews.
* Consider accessibility and user experience.
* Maintain the project's frontend-only architecture.

## Reporting Issues

If you experience or observe unacceptable behavior, report it through the project's issue tracker or contact the maintainers.

## Enforcement

Project maintainers may remove content, reject contributions, or restrict participation when behavior violates this code of conduct.

Thank you for helping make EcoTrack a positive and productive project.
