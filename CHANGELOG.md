# Changelog

All notable changes and improvements to EcoTrack will be documented in this file.

The format follows a simple version-based changelog structure.

---

## [1.0.0] - 2026-07-10

### Added

* Initial release of EcoTrack.
* Multi-step carbon footprint audit wizard.
* Transport, housing, diet, and consumption tracking.
* Real-time carbon calculation engine.
* Eco-Score generation system.
* Animated HTML5 Canvas visualizations.
* Carbon reduction mitigation simulator.
* Interactive sustainability action filters.
* Browser-based pledge wall using localStorage.
* Responsive frontend layout.
* Accessibility-focused semantic HTML structure.

### Technology

* HTML5
* CSS3
* Vanilla JavaScript ES6+
* HTML5 Canvas API
* Local Storage API

### Notes

* EcoTrack currently operates fully on the client side.
* No backend server or external database is required.
